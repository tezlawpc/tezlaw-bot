How filing actually works

Read before assuming integration exists

There is no public API for filing documents into the federal courts. The Administrative Office of the U.S. Courts exposes only the PACER Case Locator search API and an XML upload spec for bankruptcy petitions. Every motion, brief, complaint, and exhibit in district court still requires a human to log into the court's CM/ECF web interface and click through the docket events. The links below are the shortest path to that interface for each court you commonly practice in.

For each court you intend to file in, you must hold an active PACER account, separately register as an e-filer with that court, and be admitted to practice before that bench. NextGen courts let you link PACER and filer credentials; CurrentGen still uses separate logins.
