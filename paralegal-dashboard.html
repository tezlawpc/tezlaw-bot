CourtListener API Token

Anonymous access: ~100 requests/hour. Free account token: 5,000 requests/hour. Register free at courtlistener.com, then go to Profile → API Token.

Add to Zara (Render)

To enable case law search in Telegram/WhatsApp JJ mode, add this environment variable to your Render service:

COURTLISTENER_TOKEN=your_token_here

Then redeploy. In JJ mode, type: "find cases on [topic]" or "verify citation [citation]"
